# Background Music Assets

Place royalty-free lo-fi tracks here (MP3 or OGG format).

Recommended: 5–10 tracks, each 60–120 seconds.
The pipeline selects randomly and avoids repeating the same track 3× in a row.

## Sourcing

Use royalty-free music from:
- [Pixabay Music](https://pixabay.com/music/)
- [Free Music Archive](https://freemusicarchive.org/)
- [Uppbeat](https://uppbeat.io/)

Ensure the license permits YouTube monetization.
